# For developer information

## Storage data

Data format - **json**.

Data on windows stored in:

```%AppData%\Roaming\modern-reader\Data```

- **books.json** - data about books.
- **notes.json** - data about notes.
- **setting.json** - app settings.

##### Not recommended to edit!
